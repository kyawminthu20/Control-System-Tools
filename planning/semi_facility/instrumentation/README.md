# Instrumentation Knowledge Base

## Purpose

This folder captures reusable instrumentation knowledge for semiconductor facility engineering.

It sits between raw manuals and system notes:

- manuals tell you what one device can do
- instrumentation notes explain how to select and apply device families
- system notes explain where those devices fit in the facility

## Core note types

- [Selection Principles](selection_principles.md)
- [Device Family Map](device_family_map.md)
- [Measurement and Alarm Strategy](measurement_and_alarm_strategy.md)
- [Instrumentation Use Matrix](semiconductor_facility_instrumentation_use_matrix.md)
- [Manufacturer and Product Family Comparison](manufacturer_product_family_comparison.md)

## Scope

Focus on:

- process and utility instrumentation
- safety-related sensing and shutdown signals
- analyzer fit for UPW, chemicals, gas, exhaust, and cleanroom service
- device-family selection logic
- failure modes, diagnostics, and maintenance burden

## Boundary

This folder is for device knowledge and application patterns, not a vendor-specific model directory.

Model-specific facts should live in `manuals/` and then link back here.
