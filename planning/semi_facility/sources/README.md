# Public Sources

## Purpose

This folder tracks where information came from before it gets normalized into reusable engineering notes.

## Workflow

1. Add the source to [Public Source Register](public_source_register.md).
2. Create a source note from [Public Resource Note Template](public_resource_note_template.md) when the source has lasting value.
3. Link that source note from the relevant system or manual note.
4. Mark whether the result is direct evidence or working inference.

## Source priority

- Public primary sources from the publisher, regulator, or standards-body summary page
- Public university or government technical material
- Public vendor manuals and datasheets
- Public secondary explanations used only to fill gaps or improve orientation

## Rule

Do not treat a source as reusable until the citation and access model are recorded.
