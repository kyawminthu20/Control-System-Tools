# Induction Motor Nameplate and Enclosures

## What this file is

This is a cleaned work note derived from the fourth embedded lesson in [motors.md](../motors.md).

Approximate source range:

- `22:08` to `28:30`

## Topic focus

This segment summarizes common induction-motor nameplate data and enclosure terminology.

## Main concepts captured

### 1. The nameplate is the basic operating-data summary

The transcript identifies these common items:

- horsepower
- torque relationship
- rated voltage
- full-load amps
- rated frequency
- rated speed
- pole count
- phase
- NEMA design letter
- insulation class
- service factor
- frame size

### 2. Rated speed is normally the nameplate speed of interest

- The lesson explicitly states that nameplates normally show rated speed rather than synchronous speed.
- Synchronous speed can then be inferred if needed.

### 3. Voltage, current, and insulation data all affect allowable use

- Rated voltage is described as the intended operating voltage with tolerance around that value.
- Full-load current and insulation class are treated as key thermal and loading indicators.
- Service factor is presented as a short-term overload allowance rather than a reason for casual oversizing.

### 4. Frame size also carries mechanical meaning

- The transcript points out that frame size includes mounting dimensions and shaft dimensions, not just a general size label.

### 5. Enclosure type indicates how the motor handles cooling and environment

The source describes:

- `ODP`: open drip-proof
- `TENV`: totally enclosed non-ventilated
- `TEFC`: totally enclosed fan-cooled
- `TEBC`: totally enclosed blower-cooled

It also briefly names other enclosure families such as washdown, explosion-proof, and hazardous-location designs.

## Working takeaway

This segment is the basic field-reference note:

- read the nameplate before assuming voltage, current, speed, or duty
- match enclosure type to the environment and cooling method
- do not treat frame and insulation details as secondary

## Important caution

This file is a transcript-derived work note. Final motor application decisions should be based on the actual nameplate, the OEM manual, and the applicable installation and protection rules.
