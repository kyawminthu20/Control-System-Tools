# EGC Definition and Effective Fault Path

## What this file is

This is a cleaned work note derived from the opening section of [types of equipment ground conductors.md](../types%20of%20equipment%20ground%20conductors.md).

Approximate source range:

- `0:10` to `5:52`

## Topic focus

This segment tries to explain what an equipment grounding conductor really is in functional terms, especially how it relates to the effective ground-fault current path.

## Main concepts captured

### 1. The transcript is uncomfortable with the wording of the definition

- The speaker argues that the formal definition can be confusing when read literally.
- The teaching approach is to explain function first, then come back to the words.

### 2. The EGC is only one part of the full fault-return path

- The transcript repeatedly distinguishes the equipment grounding conductor from the entire effective ground-fault current path.
- The full return path includes more than the green wire or metal raceway alone.

### 3. Fault current returns to the source, not to the earth

- A major teaching point in this segment is that line-to-ground fault current is expected to return to the source through the bonding/fault-return path.
- The earth is not treated as the normal fault-clearing path.

### 4. The EGC bonds metal parts together

- The transcript frames the EGC as the conductive path that bonds exposed metal parts of equipment together as part of the return path.
- The larger path then continues through bonding jumpers and the source-side connection.

### 5. The purpose is fault clearing

- The EGC exists so a ground fault can create enough current on a low-impedance path to operate the overcurrent protective device.

## Working takeaway

This segment is best read as a functional definition:

- the EGC is part of the fault-return circuit
- it bonds metal parts together
- it must connect into a complete low-impedance path back to the source
- its purpose is operation of the protective device
