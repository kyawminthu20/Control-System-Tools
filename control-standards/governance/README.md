# Governance

**AI_READ_ACCESS: ALLOWED**

This folder defines the human processes for promoting content into `../rag/`, managing design changes, and tracking decisions.

## Scope

- promotion checklists
- design change policies
- decision logging
- release notes

## Working Rules

1. Authoritative content belongs in `../rag/`.
2. Work-in-progress belongs in `../work/`.
3. Copyrighted or proprietary source material belongs in `../restricted/`.
4. Significant process decisions should be logged here.
