# Motor Nameplate Review Checklist

## 0. Use

Use this checklist before setting overloads, selecting a starter, approving a replacement motor, or connecting a drive.

## 1. Electrical basics

- [ ] Rated voltage matches the intended supply or connection arrangement
- [ ] Frequency matches the intended system
- [ ] Phase matches the supply
- [ ] Full-load current is captured for protection review

## 2. Mechanical and thermal basics

- [ ] Rated speed is suitable for the load
- [ ] Frame size fits the mechanical arrangement
- [ ] Insulation class is recorded
- [ ] Service factor is understood but not used casually as spare capacity

## 3. Environmental fit

- [ ] Enclosure type suits the environment
- [ ] Cooling method is compatible with how the motor will run
- [ ] Washdown, dust, or hazardous-location needs are reviewed separately

## 4. Integration checks

- [ ] Starting method is confirmed
- [ ] VFD use, if any, is confirmed
- [ ] Overload setting basis is recorded
- [ ] Local disconnect/isolation expectations are known
