# Motor Selection Workflow

## 0. Purpose

Use this workflow to structure first-pass motor selection before detailed protective-device and drive coordination work begins.

## 1. Define the load first

Collect:

- load type
- required running speed
- required starting torque
- duty cycle
- stopping method

Motor selection is unreliable if started from horsepower alone.

## 2. Confirm supply and control architecture

Check:

- available voltage and frequency
- phase availability
- direct-on-line, soft-start, or VFD approach
- disconnecting and service-isolation expectations

## 3. Review environment

Check:

- ambient temperature
- washdown, dust, or corrosive exposure
- indoor or outdoor mounting
- hazardous-location implications if any

## 4. Review nameplate and construction fit

Confirm:

- power rating
- full-load current
- speed
- enclosure type
- service factor
- frame size
- insulation class

## 5. Review protection and integration

Before release, confirm the design path for:

- branch-circuit protection
- overload protection
- grounding/bonding
- drive compatibility if a VFD is used
