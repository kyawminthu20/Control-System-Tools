# Design Framework — Motor Systems

This folder holds practical motor-system review notes that bridge nameplates, supply matching, drive integration, and protection decisions.

Use it for:

- motor selection workflow
- nameplate review
- star/delta and supply matching
- VFD-to-motor integration review
- VFD and servo commissioning workflow planning
- first-pass motor and drive troubleshooting logic
- comparison-based motor selection at the concept stage
- integrated drive-on-motor architecture review
- thermal, EMC, serviceability, and standards-routing notes for integrated drives
