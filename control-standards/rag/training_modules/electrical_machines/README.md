# Training Modules — Electrical Machines

This folder holds practical motor-learning modules that support machine design, panel design, and field troubleshooting.

Use it for:

- induction motor fundamentals
- DC motor construction basics
- motor nameplate, slip, torque, and enclosure concepts
- VFD architecture and control basics
- servo-drive fundamentals and feedback concepts
- motor family comparison and AC/DC comparison
- broader comparison context for BLDC, EV traction, and drone propulsion topics
