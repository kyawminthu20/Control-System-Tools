# Training Modules — Fundamentals

This folder holds practical learning material that teaches core electrical and control-system concepts without trying to act as the standards source itself.

Use it for:

- simple explanations of theory used in panel and machine design
- step-by-step engineering logic
- worked reasoning patterns that support standards application

When a topic has a standards anchor, the training note should point back to the authoritative files under `../../standards_intelligence/`.
