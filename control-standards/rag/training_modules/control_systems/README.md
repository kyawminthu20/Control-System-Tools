# Training Modules - Control Systems

This folder holds training notes for practical feedback-control concepts used in industrial automation and motion systems.

Use it for:

- feedback-loop vocabulary
- PID control intuition
- industrial PLC PID implementation patterns
- control-loop architecture patterns for VFDs, servos, and process loops
- heater-control designs that use time-proportioning with binary outputs
- drone flight-control examples with nested rate, attitude, altitude, and position loops

Suggested reading order:

1. `pid_control_intuitive_foundation.md` - overview and reading map
2. `pid_control_intuition.md` - intuition and mental model
3. `industrial_pid_implementation.md` - PLC implementation details
4. `industrial_control_loop_architectures.md` - VFD, servo, and process-loop context
5. `pid_heater_control_with_contactor.md` - practical heater-control example
6. `pid_drone_control.md` - drone stabilization and flight-control example

When a topic becomes compliance-relevant, route back to `../../standards_intelligence/`.
