# Commissioning Checklists

This module holds field-facing verification notes that translate design intent into structured checks before energization and during initial startup.

Use it for:

- pre-power verification
- motor startup checks
- drive startup checks
- simple electrical sanity checks
- staged energization and incoming-supply review

These notes support commissioning work but do not replace OEM procedures or site safety process requirements.
