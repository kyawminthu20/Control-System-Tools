## Baseline Interpretation

- For a US-only CNC machine build, local baseline support is strongest for `NFPA 79 + NEC + UL 508A overlap guidance`. [LOCAL: crosswalks/overlap_matrix/ul508a_nec_nfpa79_overlap.md#Use Cases]
- For an international or dual-market CNC machine, local baseline support is strongest for `IEC 60204-1 + NFPA 79 + NEC`, with functional safety and risk assessment routed to non-populated modules that must be externally verified. [LOCAL: crosswalks/overlap_matrix/nfpa79_iec60204_overlap.md#Dual-Compliance Strategy] [LOCAL: _index.yaml#regional_routing]
- For software lifecycle, safety partitioning, and secure-development questions, start with the local software standards guide and then close the detailed methods against the purchased standards. [LOCAL: reference_models/software_safety_and_intrinsic_safety_standards.md#Fast Routing] [TO VERIFY: IEC 61508-3]
- For machine-tool-specific door unlocking, tool changer safeguarding, and workholding integrity requirements, use this package only as an electrical and machine-control baseline and complete the type-C machine-tool layer externally. [TO VERIFY: ISO 16090-1]
