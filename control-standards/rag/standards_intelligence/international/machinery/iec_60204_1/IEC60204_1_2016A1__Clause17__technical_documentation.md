Clause 17 of **IEC 60204-1** (2016+AMD1:2021) specifies the **technical documentation** supplied with the electrical equipment of the machine. Marking and reference designations are a separate clause — see Clause 16.

The documentation is what makes the installed machine maintainable: it is the only durable record of how the electrical equipment is intended to work, and it is the basis on which the machine is verified (Clause 18), commissioned, and later modified.

---

## 0. Scope

This clause specifies what information must be supplied with the electrical equipment, so that the machine can be safely installed, commissioned, operated, and maintained.

## 1. Documentation Set

Supplied in a medium agreed with the user.

* **Electrical schematics / circuit diagrams:** show the functional relationship of the electrical equipment, drawn with standard graphical symbols (IEC 60617) and carrying the reference designations applied under Clause 16.
* **Instruction manual:** procedures for safe commissioning and operation, plus a description of the safety functions and their maintenance and testing intervals.
* **Parts list:** the components installed, explicitly identifying **safety-related parts**, so that replacements preserve the safety integrity of the design.
* **Installation requirements:** the external supply specification, protective-earthing requirements, and the environmental conditions the equipment assumes.

## 2. Why it is safety-relevant

A safety function's integrity claim depends on the specified components and the specified proof-test/maintenance intervals. If the parts list does not flag safety-related parts, a like-for-like-looking substitution silently invalidates the PL or SIL claim. The documentation is therefore a safety control, not an administrative afterthought.

## 3. Cross-References

* **Clause 16** — marking, warning signs, reference designations (the schematics must match them).
* **Clause 18** — verification.
* US equivalent — NFPA 79 technical documentation requirements.
* EU — the technical documentation feeds the technical file required by EU machinery legislation.
