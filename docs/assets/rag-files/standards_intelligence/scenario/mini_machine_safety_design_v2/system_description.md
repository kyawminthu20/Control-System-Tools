# System Description

## Functional concept

The machine concept is a compact fluid-handling and positioning skid with two interacting domains:

- a hydraulic domain that powers a clamp circuit and a lift or index circuit
- a chemical dosing domain that meters two liquids into a process point or mix vessel

The machine is controlled from a local HMI and a PLC platform with separate safety logic. Data is logged locally and can be forwarded to a site historian through a segmented network path. Safety architecture assumes at least one machine E-stop chain, one guard interlock chain, one hydraulic pressure safety interlock, and one chemical spill or overfill shutdown path.
