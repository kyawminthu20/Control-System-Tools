# Commercial Overlay

## What changes vs baseline

- The commercial variant can usually stay closest to the baseline package, with strong emphasis on AHJ acceptance, serviceability, clear labeling, and installer-friendly documentation. [LOCAL: us/nec/NEC_2023__Art110__requirements_for_electrical_installations.md#1. Installation & listing requirements (field rules)] [LOCAL: us/nec/NEC_2023__Art409__industrial_control_panels.md#5. Inspector expectations]
- If a listed panel is preferred for insurance or customer requirements, UL 508A attention increases even if the machine function stays simple. [LOCAL: routing/standards_applicability.md#US Market Only] [LOCAL: us/ul_508a/UL508A_OVERVIEW.md#When UL 508A Applies]
- Documentation should stay concise but complete: nameplate, schematics, IO list, lockout map, and operator instructions. [LOCAL: us/nfpa79/NFPA79_2024__Ch19__marking_and_documentation.md#2. Documentation Expectations]

## Standards to prioritize

- NEC is the primary local installation code basis. [LOCAL: routing/standards_applicability.md#US Market Only]
- UL 508A becomes important when listing is requested. [LOCAL: us/ul_508a/UL508A_OVERVIEW.md#When UL 508A Applies]
- NFPA 79 remains useful if the skid is treated as industrial machinery rather than a standalone panel. [LOCAL: routing/standards_applicability.md#Applicability Matrix by Project Type]

## Typical inspection or acceptance artifacts

- Panel nameplate and SCCR basis. [LOCAL: us/nec/NEC_2023__Art409__industrial_control_panels.md#3. Labeling requirements]
- Torque verification and installation evidence. [LOCAL: us/nec/NEC_2023__Art110__requirements_for_electrical_installations.md#1. Installation & listing requirements (field rules)]
- Basic operator and maintenance instructions. [LOCAL: us/nfpa79/NFPA79_2024__Ch19__marking_and_documentation.md#2. Documentation Expectations]
- AHJ or field-evaluation checklist if no listing mark is used. [LOCAL: us/nec/NEC_2023__Art409__industrial_control_panels.md#4. Relationship to UL 508A]
