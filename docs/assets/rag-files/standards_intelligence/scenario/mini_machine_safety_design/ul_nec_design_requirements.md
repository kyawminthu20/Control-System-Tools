# UL / NEC Design Requirements

## Local-Corpus Constraint

The local corpus provides strong NEC, NFPA 79, and IEC 60204-1 content, but the UL 508A clause files are mostly draft templates. This file therefore separates:

- concrete NEC/NFPA 79/IEC-supported requirements
- UL-oriented applicability and evidence requirements
- detailed UL construction items that remain `NOT FOUND IN LOCAL CORPUS – TO VERIFY` [LOCAL: us/ul_508a/UL508A_OVERVIEW.md#content-development-status]
