## Baseline Selection Notes

- For a US-only machinery build, use NFPA 79 plus NEC Articles 670, 409, 250, 240, 300, 310, 430, and 725 as the baseline standards set. [LOCAL: routing/standards_applicability.md#general-industrial-machinery] [LOCAL: _standards_map.md#market-based-selection]
- For an international or dual-market build, use IEC 60204-1 as the electrical implementation layer and add ISO 12100 plus ISO 13849-1 or IEC 62061 for safety functions. [LOCAL: international/machinery/iec_60204_1/IEC60204_OVERVIEW.md#usage-workflow] [LOCAL: crosswalks/overlap_matrix/nfpa79_iec60204_overlap.md#safety-framework]
- For software lifecycle, redundancy, and intrinsically safe interface questions, start with the local software standards guide and then close the detailed methods against the purchased standards. [LOCAL: reference_models/software_safety_and_intrinsic_safety_standards.md#Fast Routing] [TO VERIFY: IEC 61508-3] [TO VERIFY: IEC 60079]
- For process-industry use cases where chemical overfill, release prevention, or shutdown integrity becomes a process-safety function, elevate the function review to IEC 61511/61508. Detailed process-safety lifecycle requirements are NOT FOUND IN LOCAL CORPUS – TO VERIFY. [LOCAL: _standards_map.md#applicability-matrix-by-project-type] [TO VERIFY: IEC 61511]
