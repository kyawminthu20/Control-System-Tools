## How to Use This Directory (and How Not To)

A manufacturer should never be selected merely because it appears here.
Typical evaluation criteria before specifying:

- Application requirements, environmental ratings, and performance needs
- Network/protocol compatibility with the rest of the architecture
- Safety certification where safety functions are involved
- Cybersecurity support: patching policy, hardening guides, PSIRT responsiveness
- Software licensing model and long-term cost
- Local support, distributor inventory, and spare-parts availability
- Product lifecycle status (avoid designing in end-of-life families)
- Your maintenance team's existing experience and installed base
- Plant standards and validation/regulatory requirements
