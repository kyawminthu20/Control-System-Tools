## Standards Influence

| Standard | Role at This Stage |
|----------|-------------------|
| **ISO 13849-1 Annex K** | Verification checklist for safety functions before commissioning |
| **IEC 62061** | Safety function verification requirements |
| **IEC 61511** | Pre-startup safety review (PSSR) for process SIS |

## Key Activities

- **Loop checks** — verify each I/O point operates correctly
- **Instrument calibration** — transmitters, sensors per specifications
- **Safety function dry run** — test each safety function without process
- **ISO 13849-1 Annex K checklist** — systematic verification of each safety function
- **Motor check** — correct rotation, protection settings

## Key Deliverable

**Pre-Commissioning Checklist:**
- Loop check records (each I/O point)
- Calibration records
- Safety function pre-test results (prior to energizing with process)
- Annex K verification status per safety function
