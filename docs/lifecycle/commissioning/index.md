## Standards Influence

| Standard | Role at This Stage |
|----------|-------------------|
| **ISO 13849-1** | Safety function validation — confirm PL is achieved |
| **IEC 62061** | SRECS validation requirements |
| **IEC 61511** | SIS commissioning and validation |
| **All applicable** | Final compliance verification |

## Key Activities

**Factory Acceptance Test (FAT):**
- Functional testing of all sequences at manufacturer facility
- Safety function testing with simulated inputs
- Documentation review and approval

**Site Acceptance Test (SAT):**
- Repeat key tests at installation site
- Integration testing with actual process / machine
- Safety function final testing with actual sensors and final elements

**Safety Validation:**
- Validate each safety function meets its PLr or SIL target
- Document validation evidence
- Complete the validation report

## Key Deliverable

**Validation and Verification (V&V) Report:**
- FAT and SAT test records
- Safety function validation results
- Final PL or SIL achieved (confirmed)
- As-built documentation
- Operator and maintenance training records
