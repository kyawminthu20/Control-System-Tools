## Standards Influence

| Standard | Role at This Stage |
|----------|-------------------|
| **NEC** | Installation of wiring, conduit, grounding at the site |
| **NFPA 79** | Machine electrical installation requirements |

## Key Activities

- Field wiring from service to machine disconnect
- Conduit and raceway installation per NEC
- Grounding and bonding at installation point (NEC Article 250)
- Machine feeder sizing (NEC Article 670 / NFPA 79)
- Panel mounting and environmental sealing
- Incoming power verification

## Key Deliverable

**Installation Record:**
- Site wiring records
- Grounding continuity verification
- Incoming voltage and phase verification
- As-built markup if field changes were made
