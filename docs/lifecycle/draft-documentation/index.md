## Standards Influence

All applicable standards require documentation. Key documentation requirements:

| Standard | Documentation Requirement |
|----------|--------------------------|
| **NFPA 79 Ch 19** | Technical documentation: drawings, diagrams, equipment list |
| **IEC 60204-1 Clause 17** | Technical documentation package (more prescriptive than NFPA 79) |
| **UL 508A Sec 12** | Panel marking and labeling |
| **ISO 13849-1** | Safety function documentation, verification plan |
| **IEC 62061** | SRECS documentation requirements |

## Key Deliverables

| Document | Standard Reference |
|----------|--------------------|
| Schematic drawings | NFPA 79 / IEC 60204-1 |
| P&ID (process applications) | — |
| Cause and effect matrix | IEC 61511 (process SIS) |
| Loop drawings | Process applications |
| Safety manual (draft) | ISO 13849-1 / IEC 62061 |
| Bill of materials | — |
| Equipment list | IEC 60204-1 Clause 17 |
