## Standards Influence

| Standard | Role at This Stage |
|----------|-------------------|
| **ISO 12100** | Defines scope of the machine / system; intended use and foreseeable misuse |

## Engineering Activities

- Define the machine or system boundary
- Document intended use and reasonably foreseeable misuse
- Identify applicable markets (US, EU, global)
- Identify initial list of potentially applicable standards
- Establish project safety objectives

## Key Deliverables

| Deliverable | Description |
|-------------|-------------|
| System description | Narrative of what the machine does |
| Operating assumptions | Process, environment, operator interaction |
| Boundary definition | What is inside and outside the machine's scope |
